insert

Defined in src/core.js:56

The insert event is triggered when a new block should be inserted. This happens when ENTER key is pressed at the beginning of a block (should insert before) or at the end of a block (should insert after).

Event Payload:

  • element HTMLElement

    The element triggering the event.

  • direction String

    The insert direction: "before" or "after".

merge

Defined in src/core.js:78

The merge event is triggered when two needs to be merged. This happens when BACKSPACE is pressed at the beginning of a block (should merge with the preceeding block) or DEL is pressed at the end of a block (should merge with the following block).

Event Payload:

  • element HTMLElement

    The element triggering the event.

  • direction String

    The merge direction: "before" or "after".

move

Defined in src/core.js:109

The move event is triggered when the user moves a selection in a block. This happens when the user selects some (or all) content in a block and an ARROW key is pressed (up: drag before, down: drag after).

Event Payload:

  • element HTMLElement

    The element triggering the event.

  • selection Selection

    The actual Selection object.

  • direction String

    The move direction: "before" or "after".

split

Defined in src/core.js:67

The split event is triggered when a block should be splitted into two blocks. This happens when ENTER is pressed within a non-empty block.

Event Payload:

  • element HTMLElement

    The element triggering the event.

  • before String

    The HTML string before the split.

  • after String

    The HTML string after the split.

switch

Defined in src/core.js:98

The switch event is triggered when the user switches to another block. This happens when TAB is pressed (move one block after) or SHIFT+TAB is pressed (move one block before).

Event Payload:

  • element HTMLElement

    The element triggering the event.

  • direction String

    The switch direction: "before" or "after".
